### Shivaの使い方

※ 日本語で書きます。母国語なので、英語化はFireFoxブラウザのGoogle翻訳などでお願いします。

```markdown
# Rubygems/sheltered-girlをインストール
heat branch overdrive takkii overdrive main

cd overdrive && yarn install

mix deps.get

iex -S mix

Shiva.elixir

# 終了は、Ctrl+C を2回押す。
```
